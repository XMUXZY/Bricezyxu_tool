# 风水录养成计算器 · AI使用说明

我需要你帮我实现一个**QQ华夏手游风水录养成计算器**，以下是工具的预期效果和数据说明。

---

## 🎯 工具预期效果

这是一个双向计算工具，支持以下两种模式：

**模式一：目标规划模式（已知起终点，求材料）**
- 输入：指定某个/某几个/全部风水点的当前星级 + 目标星级
- 输出：需要消耗的各类材料数量明细（按材料种类分类汇总）

**模式二：资源评估模式（已知材料，求可达等级）**
- 输入：指定某个/某几个/全部风水点的当前星级 + 持有的某种材料数量
- 输出：该材料最多能将哪些风水点升到多少星，剩余材料多少

两种模式都需要支持**单点查询**（只算某一个风水点）和**整图查询**（一次算整张宝图全部6个风水点）。

## 📊 如何阅读附件 Excel

Excel 共 6 个 Sheet，建议按以下顺序阅读：

### ① 先看「AI计算参考数据」（第6个Sheet）— 核心

每行代表一个风水点（共24行），关键列含义：

| 列名 | 含义 |
|------|------|
| `主材料每次量` | 每次操作消耗的主材料数量 |
| `副材料每次量` | 每次操作同时消耗的副材料数量（0 = 无副材料） |
| `1星~5星累计进度` | 达到该星级所需的**累计进度总量**（0星 = 0起点） |
| `卡点星级` | 该风水点在哪个星级有卡点（0 = 无卡点，北郡各点在1星有卡点） |
| `卡点主材额外量` | 突破该卡点需要在正常消耗之外**额外**支付的主材数量 |

**核心计算公式：**

```
升级 X星 → Y星 所需主材 = (Y星累计进度 - X星累计进度) × 主材每次量 + 卡点额外量（若跨越卡点）
升级 X星 → Y星 所需副材 = (Y星累计进度 - X星累计进度) × 副材每次量
```

### ② 再看「逐星级消耗明细」（第3个Sheet）— 用于验证

已预先算好每个风水点各段的材料消耗量，可用于验证计算逻辑是否正确。

### ③「宝图满星总材料汇总」（第4个Sheet）— 校验基准

各宝图从0星全部升到5星的总消耗，可作为整图计算结果的校验基准：

| 宝图 | 主材料 | 主材总量 | 副材料 | 副材总量 |
|------|-------|---------|-------|---------|
| 北郡 | 神兽石 | 7,680 | 无 | - |
| 琅琊盆地 | 缠山图 | 12,060 | 神兽石·星 | 16,080 |
| 昆仑 | 神兽石·月 | 19,000 | 寻龍图 | 15,200 |
| 轩辕 | 神兽石·日 | 45,120 | 镇龍锁 | 22,560 |

### ④「宝图与材料总表」（第2个Sheet）— 参考用

供参考各风水点所属宝图和材料配置，展示风水点名称时使用。

## ⚠️ 注意事项

1. **进度值是累计值**：「1星进度=250」的含义是"从0进度累计到250时升为1星"，而非"升1星需要250步"
2. **北郡卡点机制**：北郡6个风水点在1星均有卡点，必须额外支付材料才能解锁继续升级（其他宝图无此机制）
3. **副材料同步消耗**：部分风水点每次升级同时消耗主材料和副材料，两者进度增量相同，只是单次消耗数量可能不同
4. **材料不通用**：不同宝图的主材料完全不同，计算时注意区分
