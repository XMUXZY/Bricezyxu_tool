## 开发技巧

### 快速测试单个页面
```python
# 在页面文件末尾添加（调试用）
if __name__ == "__main__":
    import customtkinter as ctk
    root = ctk.CTk()
    root.geometry("900x700")
    
    colors = {
        "nav_bg": "#1a1a2e",
        "text": "#e0e0e0",
        "text_dim": "#888888",
        # ... 其他颜色
    }
    
    page = YourPageClass(root, colors)
    page.pack(fill="both", expand=True)
    
    root.mainloop()
```
