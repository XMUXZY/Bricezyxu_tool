## ✨ 修正内容

### 1. Tab1：根据请神符计算星级

**输入**：
- 神仙谱套组选择
- 请神符数量（新增）
- 各守护神当前星级

**计算逻辑**：
```
总残念 = 请神符数量 × 10 （重复出货转化）
每个守护神分配 = 总残念 / 守护神数量
可达星级 = 当前星级 + 残念可提升的星数
```

**输出表格**：
| 列名 | 说明 |
|------|------|
| 守护神 | 守护神名称 |
| 当前星级 | 玩家输入的起始星级 |
| 分配残念 | 平均分配到的残念数量 |
| 可达星级 | 可提升到的目标星级 |
| 剩余残念 | 升星后剩余残念 |
| 进度 | X/5星进度显示 |

**底部统计**：
- 套组总进度: X/Y星
- 使用请神符: X个
- 总残念获得: X个

---

### 2. Tab2：根据目标计算请神符

**输入**：
- 神仙谱套组选择
- 各守护神当前星级和目标星级

**计算逻辑**：
```
需要残念 = Σ(目标星级 - 当前星级的消耗)
理想请神符 = ⌈需要残念 / 10⌉ （向上取整）
残念精华 = 需要残念 × 单价（备选方案）
```

**输出表格**：
| 列名 | 说明 |
|------|------|
| 守护神 | 守护神名称 |
| 当前→目标 | 星级提升路径 |
| 需要残念 | 升星所需残念总数 |
| 理想请神符 | 假设全部重复出货的请神符数 |
| 残念阁精华 | 备选方案：用精华兑换残念 |

**底部统计**：
- 总残念需求: X个
- 理想请神符总需求: X次
- 或使用残念精华: X点

**说明**：
- 理想请神符：假设每次都重复出货转化10残念
- 实际需求会因概率浮动
- 残念精华：可通过残念阁兑换残念

## 🎮 游戏机制说明

### 残念获取途径
1. **请神重复出货**：重复召唤已有守护神 → 自动转化为10个残念
2. **残念阁兑换**：用残念精华购买指定守护神残念
3. **活动奖励**：限时活动、英雄谱等

### 升星消耗（全守护神通用）
| 目标星级 | 消耗残念 | 累计残念 |
|---------|---------|---------|
| 0→1星 | 20个 | 20 |
| 1→2星 | 40个 | 60 |
| 2→3星 | 80个 | 140 |
| 3→4星 | 160个 | 300 |
| 4→5星 | 0个 | 300（自动满星）|

### 计算公式
- **Tab1**: `总残念 = 请神符 × 10`
- **Tab2**: `请神符 = ⌈总残念 / 10⌉`

> **注意**：这里的计算是理想情况（假设全部重复出货），实际游戏中会受首次出货概率影响。

## 📊 使用示例

### 示例1：我有100个功曹请神符，能练到几星？
1. 切换到`📊 根据请神符计算星级`
2. 选择"四值功曹"套组
3. 输入请神符数量: `100`
4. 设置各守护神当前星级（如全0星）
5. 点击"🚀 计算可达星级"

**结果**：
- 总残念获得: 1000个
- 每个守护神分配: 250个
- 可达星级: 约4星（300残念才能5星）

---

### 示例2：我想把日直六星全练到5星，需要多少请神符？
1. 切换到`🎯 根据目标计算请神符`
2. 选择"日直六星"套组
3. 设置当前星级: 全0星，目标星级: 全5星
4. 点击"🎯 计算所需材料"

**结果**：
- 总残念需求: 1800个（6神×300）
- 理想请神符需求: 180次
- 或使用残念精华: 具体看各神精华价格
