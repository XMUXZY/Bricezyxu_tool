## 系统概述

**装备注灵**是经典区装备的随机属性强化系统。玩家消耗专属材料，为装备各部位注入随机属性值。每次注灵随机抽取属性值（范围内均匀分布），属性值分为**绿/蓝/紫/橙/红**五档品质。

### 核心机制

1. **首次开启**：消耗开启材料，为某部位的某等级首次注灵
2. **重刷属性**：消耗重刷材料，重新随机抽取属性值，追求更高品质（通常目标是红色）
3. **保底材料**：重刷时可选用保底材料（如新月之石），提升获得高品质的概率，减少期望次数

### 重要概念

- **装备部位数**：每套注灵覆盖11个装备部位（胸/头/腿/颈/腕/右手武器/左手武器/手指/背部/腰部/脚部）
- **重刷期望次数**：使用保底材料时，重刷到红色属性的平均期望次数（本文档所有数据均为**带保底材料**的期望值）
- **8套注灵体系**：神农→女娲→伏羲→轩辕→鸿钧→混元→夸父→蚩尤，随角色等级解锁

## 计算器使用说明（给AI）

配套 Excel 文件包含4个Sheet：

### Sheet1: AI计算数据_机器可读

**字段说明：**
- `套装名称`：神农/女娲/伏羲/轩辕/鸿钧/混元/夸父/蚩尤
- `解锁角色等级`：角色需达到的等级
- `注灵等级`：当前注灵等级（1-10）
- `装备部位数`：固定11个
- `开启材料名称/ID/单部位数量`：首次开启该等级单个部位的消耗
- `开启全部位总数量`：= 单部位数量 × 11
- `累计开启总量(0到本级)`：从1级开启到本级的累计主材料消耗
- `重刷材料1名称/ID/单次数量`：每次重刷单个部位的主材料消耗
- `重刷材料2名称(保底材料)/ID/单次数量`：每次重刷单个部位的保底材料消耗
- `重刷期望次数`：使用保底材料时，重刷到红色的期望次数（固定6次）
- `单级重刷材料1总消耗`：= 单次数量 × 11部位 × 期望次数
- `累计重刷材料1总消耗`：从1级重刷到本级的累计主材料消耗
- `单级重刷材料2总消耗`：= 保底材料单次数量 × 11部位 × 期望次数
- `累计重刷材料2总消耗`：从1级重刷到本级的累计保底材料消耗

### 计算逻辑

#### 1. 计算某套从X级升到Y级需要的材料
```
筛选条件：套装名称 = 目标套装 AND 注灵等级 >= X AND 注灵等级 <= Y
开启消耗 = SUM(开启全部位总数量)  # 如果X=1，否则为0
重刷消耗 = SUM(单级重刷材料1总消耗)
保底材料 = SUM(单级重刷材料2总消耗)
```

#### 2. 计算单部位从X级升到Y级需要的材料
```
材料消耗 = 全部位消耗 / 11
```

#### 3. 计算某套满级（10级）总消耗
```
直接查该套装 注灵等级=10 的行：
  开启累计 = 累计开启总量(0到本级)
  重刷累计 = 累计重刷材料1总消耗
  保底累计 = 累计重刷材料2总消耗
  总主材料 = 开启累计 + 重刷累计
```
