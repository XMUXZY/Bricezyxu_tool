> **适用版本：** 经典区（非百级区）

> **生成日期：** 2026-04-22

---

## 一、什么是守护神？

守护神是QQ华夏的战力养成系统之一。玩家通过消耗**请神符**召唤守护神，召唤后可通过**供奉（拜神）**消耗守护神对应的**残念**道具，将守护神的星级从1星提升至5星，从而解锁**神仙谱**套装效果，全面提升攻击力、生命等角色属性。

## 三、核心养成流程

### 3.1 第一步：获得守护神（请神）

使用对应**请神符**（消耗道具）召唤守护神，可获得对应神仙谱组内某一守护神：

- 每次请神消耗1个对应请神符
- 召唤为随机（按权重抽取，高品质权重低）
- 支持幸运值保底：积累幸运值（每次=100积分），达到上限时必得稀有守护神
- 重复获得已有守护神时，将转化为**10个对应残念**道具

### 3.2 第二步：提升星级（供奉/拜神）

持有守护神后，通过消耗**该守护神对应的残念**道具提升星级：

| 目标星级 | 消耗残念数量 |
|---------|-----------|
| 提升至 1星 | 20个 |
| 提升至 2星 | 40个 |
| 提升至 3星 | 80个 |
| 提升至 4星 | 160个 |
| 提升至 5星 | 0个（满星无需消耗）|

> **注意：** 每个守护神有独立的残念道具（如「值时功曹残念」ID: 59400237），不可混用。

**单个守护神从0→5星总残念消耗：**  
20 + 40 + 80 + 160 = **300个残念**

## 四、残念的来源（道具获取途径）

残念是供奉的核心消耗品，有以下获取途径：

| 获取方式 | 说明 |
|---------|-----|
| **请神重复获得** | 重复请神召唤到已有守护神时，自动转化为该守护神残念×10 |
| **残念阁购买** | 用**残念精华**兑换指定守护神残念 |
| **元宝/银两商城** | 部分组的请神符可通过商城直购（元宝/银两） |
| **氏族贡献商城** | 功曹请神符可通过贡献商城获得（参数10079） |
| **云霄铺** | 部分请神符可在云霄铺中购买 |
| **限时活动** | 活动奖励中包含请神符或残念 |
| **英雄谱奖励** | 英雄谱积分达标后可兑换残念/请神符 |

### 4.1 残念精华机制（低级残念→高级残念转化）

**残念精华**（ID: 59400254）是跨守护神通用货币，是守护神养成的核心资源转化路径：

**典型流程：**
1. 玩家大量请神，重复出货低品质守护神（如值时功曹）→ 获得大量低品质残念
2. 低品质残念在**残念阁**出售/分解 → 换取**残念精华**
3. 用残念精华在**残念阁购买高品质守护神残念**（如日游神、湘夫人、阴光神等）

## 七、材料规划（单守护神全满星估算）

**每个守护神从0→5星所需残念数量固定为300个。**

| 目标 | 所需请神符（仅靠请神获残念） | 说明 |
|------|--------------------------|-----|
| 单个守护神5星 | 约30次请神（若均重复出货，各得10残念）| 实际受随机出货率影响极大 |
| 单套组（4神）全5星 | 约120次请神（理想） | 每神各300残念，共1200残念 |
| 含高品质4神套组 | 明显偏高 | 高品质出货率低，需更多次数 |

> **推荐策略：** 低品质守护神优先靠请神重复获得残念；高品质守护神（紫色/金色）建议通过残念精华兑换补量，以减少请神符浪费。

## 八、计算器使用说明

配套 Excel 计算器（`守护神养成计算器数据包.xlsx`）支持：

- **模式1（材料→进度）：** 输入持有的请神符数量 + 各守护神当前星级 → 计算可推进到哪个星级
- **模式2（目标→材料）：** 输入目标星级 → 计算所需各类请神符和残念数量
