table_data = []

## 📊 表格设计原则

### 1. 材料名称与数量分列 ⭐⭐⭐

**错误示例❌**：
```python
table_data.append([
    "Lv.1",
    "青暗紫砂 × 100",  # 混合在一列，无法在Excel中计算
    "✅"
])
```

**正确示例✅**：
```python
table_data.append([
    "Lv.1",
    "青暗紫砂",  # 材料名称
    100,          # 纯数字，可计算
    "✅"
])
```

### 2. 关键信息独立列

将重要信息拆分为独立列，方便筛选和分析：

```python
headers = [
    "等级",
    "材料名称",    # 独立列，可筛选
    "材料数量",    # 独立列，可求和
    "成功率",      # 独立列，可排序
    "备注",        # 额外信息
    "状态"
]
```

### 3. 汇总信息放在表格底部

```python
# 添加空行分隔
table_data.append(["", "", "", "", "", ""])

# 添加汇总行
table_data.append([
    "总计",
    material_name,
    total_qty,  # 总数量
    "",
    "",
    "汇总"
])
```

### 4. 备注信息独立列

**改造前**：
```python
"青暗紫砂 × 100 (差50)"  # 备注混在数据中
```

**改造后**：
```python
["青暗紫砂", 100, "(差50)"]  # 备注独立一列
```

## 📝 改造模板

### 典型计算器改造模板

```python
def _calc_function(self):
    try:
        # 1. 获取输入参数
        start_lv = int(self.entry_start.get() or "0")
        target_lv = int(self.entry_target.get() or "10")
        
        # 2. 数据验证
        if start_lv >= target_lv:
            self._show_error(self.result_box, "目标等级必须大于起始等级")
            return
        
        # 3. 构建表格数据
        table_data = []
        total = 0
        
        for lv in range(start_lv + 1, target_lv + 1):
            # 计算逻辑...
            mat_qty = lv * 10
            total += mat_qty
            
            table_data.append([
                lv,              # 等级
                "材料名",        # 材料名称
                mat_qty,         # 数量（纯数字）
                "✅ 达成"        # 状态
            ])
        
        # 4. 添加汇总行
        table_data.append(["", "", "", ""])  # 空行
        table_data.append([
            "总计",
            "材料名",
            total,
            "汇总"
        ])
        
        # 5. 显示表格
        headers = ["等级", "材料名称", "材料数量", "状态"]
        title = f"计算结果 (Lv.{start_lv}→Lv.{target_lv})"
        show_table_result(self, title, table_data, headers)
        
        # 6. 显示摘要
        summary = f"✅ 计算完成\n\n"
        summary += f"起始: Lv.{start_lv}\n"
        summary += f"目标: Lv.{target_lv}\n"
        summary += f"总消耗: {total}\n\n"
        summary += "详细结果已显示在表格窗口中"
        self._show_result(self.result_box, summary)
        
    except Exception as ex:
        self._show_error(self.result_box, f"计算出错: {ex}")
```

## ⚠️ 注意事项

### 1. 数据类型

- ✅ 数量列使用 `int` 或 `float`（可计算）
- ❌ 避免字符串混合数字：`"100颗"` → 应该是 `100`

### 2. 换行处理

- 如果单元格需要换行，使用 `\n`：
  ```python
  f"{material1} × {qty1}\n{material2} × {qty2}"
  ```
- 但更推荐**分列**而不是换行

### 3. 空值处理

- 空列使用空字符串 `""`
- 不要使用 `None`（会显示为"None"）

### 4. 表头命名

- 简洁明了：`"等级"` 而不是 `"目标等级(Lv)"`
- 避免过长：控制在10个字符以内
- 统一格式：所有表头使用相同风格

### 5. 窗口置顶

**重要**：表格组件已默认置顶，无需额外配置。
- `attributes("-topmost", True)` 已在 `utils/table_result.py` 中设置
- `lift()` 和 `focus_force()` 确保窗口在最前

## ❓ 常见问题

### Q1: 表格数据过多，如何处理？

A: 表格组件自动支持滚动条，无需担心数据量。可以添加分页或筛选功能。

### Q2: 如何在表格中显示公式？

A: 导出Excel后，在Excel中添加公式。表格组件专注于数据展示。

### Q3: 如何自定义表格样式？

A: 修改 `utils/table_result.py` 中的颜色和字体参数。

### Q4: 可以支持图片或图表吗？

A: 当前版本仅支持文本数据，图表需在Excel中生成。
