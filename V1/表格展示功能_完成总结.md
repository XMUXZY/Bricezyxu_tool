## 🌟 核心特性

- 网格布局，清晰易读

### ✅ 导出CSV文件
- UTF-8编码（兼容Excel）
- 可用Excel、记事本、数据分析工具打开

## 🚀 使用方法

### 快速上手（3行代码）

```python
from utils.table_result import show_table_result

# 准备数据
data = [
    ["Lv.1", "100", "50", "✅"],
    ["Lv.2", "200", "100", "✅"],
]
headers = ["等级", "材料1", "材料2", "状态"]

# 显示表格
show_table_result(self, "计算结果", data, headers)
```

### 支持的数据格式

#### 格式1：二维列表 + 表头
```python
data = [
    ["值1", "值2", "值3"],
    ["值4", "值5", "值6"],
]
headers = ["列1", "列2", "列3"]
show_table_result(self, "标题", data, headers)
```

#### 格式2：字典列表（自动提取表头）
```python
data = [
    {"姓名": "张三", "年龄": 25, "分数": 90},
    {"姓名": "李四", "年龄": 30, "分数": 85},
]
show_table_result(self, "标题", data)  # headers 自动生成
```

## ❓ 常见问题

### Q1: 导出Excel时提示缺少依赖？
**A:** 首次使用会自动提示安装 `openpyxl`，点击确认即可。  
或手动安装：`pip install openpyxl`

### Q2: 复制后粘贴到Excel没有对齐？
**A:** 确保使用的是 **Ctrl+V** 粘贴，不要使用右键菜单粘贴。  
如果还不行，尝试选中A1单元格后粘贴。

### Q3: 表格显示不全怎么办？
**A:** 窗口支持滚动，鼠标滚轮或拖动滚动条即可查看。  
也可以导出为Excel文件查看完整数据。

### Q4: 如何自定义表格样式？
**A:** 编辑 `utils/table_result.py` 文件，修改颜色、字体等参数。  
详见文件中的注释说明。
