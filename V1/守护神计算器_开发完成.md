### 3. 精确的升星计算
```python
def _calculate_achievable_star(self, current_star, relic_count):
    """计算持有残念可达的星级"""
    star = current_star
    remaining = relic_count
    
    while star < 5:
        next_cost = int(self.upgrade_costs[str(star + 1)])
        if next_cost == 0:  # 4→5无需消耗
            star = 5
            break
        if remaining >= next_cost:
            remaining -= next_cost
            star += 1
```

## 🚀 使用流程

### 方式1: 评估可达星级
1. 点击导航栏"守护神养成"🛡️
2. 选择"📊 根据残念计算星级"标签
3. 在下拉菜单选择神仙谱套组
4. 为每个守护神输入当前星级和持有残念数量
5. 点击"🚀 计算可达星级"
6. 查看表格结果(可导出Excel)

### 方式2: 规划养成材料
1. 切换到"🎯 根据目标计算材料"标签
2. 选择神仙谱套组
3. 为每个守护神设置当前星级和目标星级
4. 点击"🎯 计算所需材料"
5. 查看残念需求、精华换算、请神符估算
