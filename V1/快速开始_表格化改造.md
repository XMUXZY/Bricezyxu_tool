## ⭐ 核心原则

### 1. 材料名称和数量必须分列

```python
# ❌ 错误
["Lv.1", "材料A × 100", "✅"]

# ✅ 正确  
["Lv.1", "材料A", 100, "✅"]
```

### 2. 数量列使用纯数字

```python
# ❌ 错误
["材料A", "100颗"]

# ✅ 正确
["材料A", 100]
```

### 3. 备注信息独立列

```python
# ❌ 错误
["材料A × 100 (差50)"]

# ✅ 正确
["材料A", 100, "(差50)"]
```

## 📋 完整示例

```python
def _calc_materials(self):
    """材料计算函数"""
    try:
        # 1. 获取输入
        start_lv = int(self.entry_start.get() or "0")
        target_lv = int(self.entry_target.get() or "10")
        
        # 2. 参数验证
        if target_lv <= start_lv:
            self._show_error(self.result_box, "目标等级必须大于起始等级")
            return
        
        # 3. 计算逻辑
        table_data = []
        total_qty = 0
        
        for lv in range(start_lv + 1, target_lv + 1):
            mat_name = "材料A"
            mat_qty = lv * 10
            total_qty += mat_qty
            
            # 构建行数据
            table_data.append([
                lv,           # 等级
                mat_name,     # 材料名称
                mat_qty,      # 材料数量（纯数字）
                "✅ 已达成"   # 状态
            ])
        
        # 4. 添加汇总行
        table_data.append(["", "", "", ""])  # 空行分隔
        table_data.append([
            "总计",
            "材料A",
            total_qty,
            "汇总"
        ])
        
        # 5. 显示表格
        headers = ["等级", "材料名称", "材料数量", "状态"]
        title = f"材料需求计算 (Lv.{start_lv}→Lv.{target_lv})"
        show_table_result(self, title, table_data, headers)
        
        # 6. 显示摘要
        summary = f"✅ 计算完成\n\n"
        summary += f"起始等级: Lv.{start_lv}\n"
        summary += f"目标等级: Lv.{target_lv}\n"
        summary += f"总消耗: {total_qty}\n\n"
        summary += "详细结果已显示在表格窗口中\n"
        summary += "可复制到Excel或导出为文件"
        self._show_result(self.result_box, summary)
        
    except Exception as ex:
        self._show_error(self.result_box, f"计算出错: {ex}")
```

## 🆘 遇到问题？

### Q: 表格窗口被遮挡？
A: 已在 `utils/table_result.py` 中修复，无需处理。

### Q: 如何添加汇总行？
A: 
```python
table_data.append(["", "", "", ""])  # 空行
table_data.append(["总计", "材料名", 总数, "汇总"])
```

### Q: 数量列无法在Excel中计算？
A: 确保使用纯数字而不是字符串：
```python
# ❌ 错误
table_data.append(["Lv.1", "材料A", "100颗"])

# ✅ 正确
table_data.append(["Lv.1", "材料A", 100])
```
