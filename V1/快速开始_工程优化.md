## 🎯 核心特性

### 1. 📦 页面基类 `BaseToolPage`

**位置**: `pages/base_tool.py`

**提供的公共方法**:

```python
from pages.base_tool import BaseToolPage

class MyToolPage(BaseToolPage):
    def _calculate(self):
        # 安全解析整数
        val = self._parse_int(self.entry.get(), 0)
        
        # 显示错误
        if val < 0:
            self._show_error(self.result_box, "数值不能为负")
            return
        
        # 显示结果
        self._show_result(self.result_box, f"结果: {val}")
        
        # 格式化数字
        formatted = self._fmt(123456.789)  # "123,456.8"
```

**可用方法清单**:

| 方法 | 功能 | 示例 |
|------|------|------|
| `_show_result(textbox, text)` | 在 textbox 显示结果 | `self._show_result(self.tb, "成功")` |
| `_show_error(textbox, msg)` | 在 textbox 显示错误 | `self._show_error(self.tb, "输入无效")` |
| `_bind_mousewheel(widget)` | 阻止滚轮事件冒泡 | `self._bind_mousewheel(self.tb)` |
| `_parse_int(val, default)` | 安全解析整数 | `num = self._parse_int("123", 0)` |
| `_parse_float_or_inf(val)` | 解析浮点数/无穷 | `f = self._parse_float_or_inf("")  # inf` |
| `_fmt(num)` | 格式化数字 | `s = self._fmt(12345)  # "12,345"` |

## ✨ 最佳实践

### 1. 创建新工具时

✅ **推荐做法**:
```python
# 使用模板 + 基类
from pages.base_tool import BaseToolPage

class MyToolPage(BaseToolPage):
    def _build_ui(self):
        # 构建界面
    
    def _calculate(self):
        val = self._parse_int(self.entry.get(), 0)
        self._show_result(self.box, f"结果: {val}")
```

❌ **不推荐做法**:
```python
# 从零开始，重复实现公共方法
class MyToolPage(ctk.CTkFrame):
    def _show_result(self, box, text):  # 重复代码
        box.configure(state="normal")
        # ...
```

---

### 2. 管理游戏数据时

✅ **推荐做法**:
- 配置数据放在 `game_data/*.json`
- 使用 `load_game_data()` 加载
- 数据更新时只修改 JSON

❌ **不推荐做法**:
- 硬编码在 Python 文件中
- 数据更新需要改源码

---

### 3. 处理用户输入时

✅ **推荐做法**:
```python
# 使用基类的安全解析方法
val = self._parse_int(entry.get(), 0)
if val < 0:
    self._show_error(box, "数值无效")
```

❌ **不推荐做法**:
```python
# 直接 int() 可能抛异常
try:
    val = int(entry.get())
except:
    # 手动显示错误
```

### Q4: 改造后程序报错怎么办？

1. 检查是否正确导入基类: `from pages.base_tool import BaseToolPage`
2. 检查类继承: `class XXX(BaseToolPage):`
3. 检查 `__init__` 调用: `super().__init__(parent, colors)`
4. 确保删除的方法没有被特殊逻辑使用
