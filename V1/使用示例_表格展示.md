# 表格化结果展示 - 使用指南

## 📋 功能特性

新增了通用的表格结果展示组件 `TableResultWindow`，提供：

✅ **表格化显示** - 网格布局，清晰易读  
✅ **一键复制** - 复制为Excel格式（制表符分隔），可直接粘贴到Excel  
✅ **导出Excel** - 生成格式化的.xlsx文件（带表头样式、自动列宽）  
✅ **导出CSV** - 生成UTF-8编码的.csv文件  
✅ **响应式设计** - 自适应窗口大小，支持滚动  

---

## 🚀 快速使用

### 方式一：使用快捷函数（推荐）

```python
from utils.table_result import show_table_result

# 在你的工具页面中调用
def show_result(self):
    # 准备数据
    data = [
        ["Lv.1", "100", "50", "✅ 已达成"],
        ["Lv.2", "200", "100", "✅ 已达成"],
        ["Lv.3", "300", "150", "❌ 材料不足"],
    ]
    headers = ["等级", "主材料", "辅材料", "状态"]
    
    # 显示表格
    show_table_result(self, "注灵计算结果", data, headers)
```

### 方式二：使用字典列表（自动提取表头）

```python
from utils.table_result import show_table_result

data = [
    {"等级": "Lv.1", "主材料": 100, "辅材料": 50, "状态": "✅"},
    {"等级": "Lv.2", "主材料": 200, "辅材料": 100, "状态": "✅"},
]

show_table_result(self, "计算结果", data)  # headers 自动从字典键提取
```

### 1. 注灵养成计算器 (tool_zhu_ling.py)

在文件顶部导入：
```python
from utils.table_result import show_table_result
```

修改 `_calc_by_material` 方法：
```python
def _calc_by_material(self):
    """根据持有材料 → 算可达等级"""
    try:
        set_name = self.t1_set_var.get()
        start_lv = int(self.t1_start_lv.get() or "0")
        
        # ... 原有验证逻辑 ...
        
        # 构建表格数据
        table_data = []
        headers = ["等级", "开启消耗", "刷新主材料", "刷新保底", "剩余材料", "状态"]
        
        reachable_lv = start_lv
        for lv in range(start_lv + 1, 11):
            cost_row = cost_data[cost_data['等级'] == lv].iloc[0]
            # ... 计算逻辑 ...
            
            if can_upgrade:
                table_data.append([
                    f"Lv.{lv}",
                    f"{open_cost_name} × {open_need}",
                    f"{refresh_main_name} × {refresh_main_total}",
                    f"{refresh_bao_name} × {refresh_bao_total}",
                    f"{open_cost_name}:{open_avail - open_need}",
                    "✅"
                ])
                reachable_lv = lv
            else:
                table_data.append([
                    f"Lv.{lv}",
                    f"{open_cost_name} × {open_need}",
                    f"差 {refresh_main_total - refresh_main_avail}",
                    f"差 {refresh_bao_total - bao_avail}",
                    "-",
                    "❌ 不足"
                ])
                break
        
        # 添加汇总行
        table_data.append([])  # 空行
        table_data.append([
            "汇总",
            f"可达 Lv.{reachable_lv}",
            f"提升 {reachable_lv - start_lv} 级",
            "",
            "",
            ""
        ])
        
        # 显示表格
        show_table_result(
            self,
            f"{set_name} - 注灵养成计算",
            table_data,
            headers
        )
        
    except Exception as e:
        messagebox.showerror("错误", f"计算失败：{e}", parent=self)
```

```python
from utils.table_result import show_table_result

def batch_parse_colors(self):
    """批量解析颜色（假设新功能）"""
    input_text = self.input_entry.get("1.0", "end-1c")
    lines = input_text.split("\n")
    
    results = []
    headers = ["原始输入", "解析颜色", "十六进制", "预览"]
    
    for line in lines:
        if not line.strip():
            continue
        # 解析逻辑...
        color_hex = self._parse_color(line)
        results.append([
            line.strip(),
            "成功" if color_hex else "失败",
            color_hex or "-",
            "█" if color_hex else "-"
        ])
    
    show_table_result(self, "批量解析结果", results, headers)
```

## 💡 使用技巧

### 1. 复制到Excel
点击 "📋 复制为Excel格式" 后：
1. 打开Excel
2. 选中A1单元格
3. Ctrl+V 粘贴
4. 数据自动按列对齐 ✅

### 2. 导出Excel文件
- 首次使用会提示安装 `openpyxl`（自动安装）
- 生成的Excel文件包含：
  - 表头样式（蓝色背景，白色加粗字体）
  - 自动调整列宽
  - 边框线

### 3. 导出CSV
- UTF-8编码（兼容Excel）
- 可用Excel/记事本/数据分析工具打开

## ⚠️ 注意事项

1. **数据类型**: 所有数据会自动转换为字符串显示
2. **空值处理**: 空值显示为空字符串
3. **特殊字符**: 支持Emoji和中文
4. **性能**: 建议单次显示不超过1000行（超过会较慢）
