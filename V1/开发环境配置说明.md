## 🎯 核心功能模块

### 1. 主框架 (`main.py`)
- **左侧导航栏**: 11个工具页面 + 设置页
- **右侧内容区**: 动态切换页面
- **主题配色**: 深色主题（#1a1a2e导航 + #0f0f1a内容）
- **页面管理**: 懒加载 + lift() 优化

### 2. 工具页面

#### 🎨 颜色解析工具 (`tool_a.py`)
- 解析游戏邮件中的 `<color=xxx>文本</color>` 标签
- 支持格式: 十六进制、RGB、RGBA、HSL、颜色名称
- 实时预览游戏邮件效果
- 22色预设色板 + 自定义颜色选择器

#### ⚙️ 淬炼计算器 (`tool_b.py`)
- 游戏装备淬炼系统计算

#### ⚔️ 重铸计算器 (`tool_c.py`)
- 装备重铸成本计算

#### ⭐ 星录/圣石/星图/风水录养成计算器
- 多维度养成系统数据分析
- 基于Excel参考数据表

#### ⚒️ 磨砺养成计算器 (`tool_gem_grind.py`)
- 宝石磨砺玩法计算

#### ⚔️ 注灵养成计算器 (`tool_zhu_ling.py`)
- **两种计算模式**:
  - 模式一: 根据持有材料 → 计算可达等级
  - 模式二: 根据目标等级 → 计算所需材料
- 8套注灵体系（神农、女娲、伏羲、轩辕、鸿钧、混元、夸父、蚩尤）
- 每套10级、全套11部位
- 保底材料重刷期望计算

### 3. 在线更新模块 (`updater.py`)
- 版本检测（从远程服务器拉取 version.json）
- 增量/全量下载
- SHA256文件校验
- 自动重启应用
- 配置服务器: `https://gitee.com/XMU_xzy/game-ops-toolkit-releases/`

### 4. 打包工具 (`build.bat`)
- 基于 PyInstaller
- 单文件 + 无窗口模式
- 自动打包为 `游戏运营工具合集.exe`

## 🔧 开发建议

### 代码规范
- 使用 CustomTkinter 组件（ctk.CTkFrame, ctk.CTkLabel等）
- 所有页面继承 `ctk.CTkFrame`
- 构造函数接收 `parent` 和 `colors` 字典
- 颜色统一使用 `self.colors` 配色方案

### 布局建议
- 使用 `CTkScrollableFrame` 作为主容器（支持长内容滚动）
- 使用 `grid()` 布局（响应式更好）
- 卡片式设计：`fg_color="#1a1a2e"`, `corner_radius=12`

### 配色方案
```python
colors = {
    "nav_bg": "#1a1a2e",      # 导航背景
    "nav_hover": "#16213e",   # 悬停色
    "nav_active": "#0f3460",  # 激活色
    "accent": "#e94560",      # 强调色
    "content_bg": "#0f0f1a",  # 内容背景
    "text": "#e0e0e0",        # 主文字
    "text_dim": "#888888",    # 次要文字
}
```
