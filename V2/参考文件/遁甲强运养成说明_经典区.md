## 系统概述

**遁甲（强运）**是经典区奇门遁甲系统的核心强化模块。通过消耗**古迹灵石**升级遁甲等级，提升能力值，解锁更高品阶。

### 品阶体系

遁甲分为4个品阶，每个品阶有独立的等级体系和专属材料：

| 品阶 | 等级范围 | 专属材料 | 材料ID |
|------|---------|---------|--------|
| **黄阶** | 1-36级 | 古迹灵石 | 59011243 |
| **玄阶** | 1-49级 | 古迹灵石·玄 | 59011248 |
| **地阶** | 1-64级 | 古迹灵石·地 | 59011249 |
| **天阶** | 1-100级 | 古迹灵石·天 | 59011250 |

### 升级机制

- **每级消耗1个材料**（低等级）
- **高等级消耗递增**：天阶后期每级消耗最多18个材料
- **品阶进阶**：满级后自动进阶到下一品阶（黄→玄→地→天）

## 品阶进阶说明

### 进阶节点

- **黄阶36级** → 自动进阶到玄阶1级
- **玄阶49级** → 自动进阶到地阶1级
- **地阶64级** → 自动进阶到天阶1级
- **天阶100级** → 满级（道阶未开放）

### 进阶规则

1. **进阶不消耗额外材料**
2. **进阶后使用新品阶材料**（古迹灵石→玄→地→天）
3. **能力值累加**：进阶后能力值在前一品阶基础上继续增长
4. **等级重新计数**：进阶后等级从1级开始（品阶内独立计数）

## 材料系统说明

### 材料独立性

4种材料**不能相互转换**，各品阶必须使用对应材料：

| 材料 | 道具ID | 适用品阶 | 获取途径 |
|------|--------|---------|---------|
| 古迹灵石 | 59011243 | 黄阶 | 活动、副本、任务 |
| 古迹灵石·玄 | 59011248 | 玄阶 | 活动、副本、任务 |
| 古迹灵石·地 | 59011249 | 地阶 | 活动、副本、任务 |
| 古迹灵石·天 | 59011250 | 天阶 | 活动、副本、任务 |

### 材料消耗特点

- **低品阶材料消耗少**：黄/玄/地阶每级固定1个
- **天阶材料消耗递增**：后期每级最高18个
- **天阶占比最大**：249/398 = 62.6% 的材料消耗在天阶

## 计算逻辑

### 1. 计算从X级升到Y级需要的材料

**跨品阶情况**需要分段计算：

```
假设从黄阶10级升到玄阶20级：
  1. 黄阶部分：黄阶10级→36级，查询SUM(消耗数量)
  2. 玄阶部分：玄阶1级→20级，查询SUM(消耗数量)
  3. 分别统计：古迹灵石 X个 + 古迹灵石·玄 Y个
```

**同品阶情况**直接累加：

```
筛选条件：品阶 = 目标品阶 AND 等级 > X AND 等级 <= Y
材料消耗 = SUM(消耗数量)
```

### 2. 根据持有材料计算可达等级

```
输入：品阶、当前等级、持有材料数量
步骤：
  1. 从当前等级开始逐级累加材料消耗
  2. 当累加量 > 持有数量时，停止
  3. 返回可达到的最高等级
  
注意：到达品阶满级后，需判断是否持有下一品阶材料
```

**示例**：
- 持有100个古迹灵石·天
- 当前天阶50级 → 逐级累加：1+1+1+...+2+2+...
- 累加到某级时超过100 → 返回该级-1

### 3. 跨品阶材料问题

**重要**：4个品阶材料**完全独立**，不能替代：

- 黄阶只能用古迹灵石
- 玄阶只能用古迹灵石·玄
- 地阶只能用古迹灵石·地
- 天阶只能用古迹灵石·天

**跨品阶升级计算**必须分段统计每种材料需求

## 常见问题

**Q1：为什么天阶消耗那么多材料？**
A：天阶等级跨度大（1-100级），且后期每级消耗递增（最高18个），因此累计消耗249个

**Q2：品阶进阶需要额外材料吗？**
A：不需要，达到品阶满级自动进阶，无额外消耗

**Q3：不同品阶材料可以通用吗？**
A：不能，4种材料各自独立，黄阶材料不能用于玄阶升级

**Q4：能力值有什么用？**
A：能力值决定遁甲提供的属性加成（具体属性由遁甲职业属性表决定）

**Q5：如何获得高品阶材料？**
A：通过游戏内高等级活动、副本、任务产出对应品阶的古迹灵石

**Q6：天阶后期消耗为什么递增？**
A：设计上后期升级难度递增，每级材料需求从1个逐步增加到18个，拉长养成周期
